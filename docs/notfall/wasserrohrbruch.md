# Wasserrohrbruch / Wasserschaden

!!! danger "Sofortmaßnahmen"
    1. **Hauptwasserhahn schließen** → siehe [Absperrhähne](../wasser/absperrhaehne.md)
    2. **Strom in betroffenem Bereich abschalten** (FI/Sicherung)
    3. **Wasser aufnehmen** – mit Lappen, Eimern, Wassersauger
    4. **Fotos für Versicherung machen**

## Wo ist der Hauptabsperrhahn?

➡️ Detaillierte Beschreibung mit Fotos: [Wasser → Absperrhähne](../wasser/absperrhaehne.md)

## Wasserschaden eingrenzen

| Bereich | Was tun |
|---------|---------|
| Apartment-WC / Bad | Eckventil unter dem WC / Waschbecken zudrehen |
| Küche | Eckventile unter Spüle zudrehen |
| Heizungsrohr | Heizung-Vorlauf/Rücklauf-Absperrung im Heizraum |
| Boiler | Kaltwasser-Zulauf zum Boiler zudrehen |

## Fachfirma kontaktieren

Siehe [Notfall-Kontakte](kontakte.md):

- Installateur (24h-Notdienst)
- Bei Rohrbruch im Mauerwerk: ggf. Trocknungsfirma

## Dokumentation für Versicherung

- [ ] Fotos vor Reinigung/Reparatur
- [ ] Schadenshergang notieren (Datum, Uhrzeit, was passiert ist)
- [ ] Schaden an Inventar listen (Möbel, Elektronik)
- [ ] Versicherung **innerhalb 7 Tagen** informieren

## Frostschäden (Winter)

Im Villgratental bei Temperaturen unter -15°C besondere Vorsicht!

➡️ Siehe [Frostschutz](../wasser/frostschutz.md) für Präventionsmaßnahmen.
