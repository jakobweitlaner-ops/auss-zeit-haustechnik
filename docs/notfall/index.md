## Allgemeine Vorgehensweise

1. **Sicherheit zuerst**: Personen in Sicherheit bringen, ggf. Strom/Wasser absperren
2. **Schaden begrenzen**: Hauptabsperrung, FI-Schalter, etc.
3. **Dokumentieren**: Fotos vom Schaden für Versicherung
4. **Gäste informieren** (falls betroffen)
5. **Fachfirma kontaktieren** → siehe [Kontakte](kontakte.md)
6. **Versicherung informieren** (innerhalb Frist)
