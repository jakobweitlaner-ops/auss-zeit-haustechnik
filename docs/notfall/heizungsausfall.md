# Heizungsausfall

## Erste Diagnose

1. **Display der Heizung prüfen**: Wird ein Fehlercode angezeigt? → [Fehlercodes](../heizung/stoerungen.md)
2. **Strom da?** → [Stromausfall](stromausfall.md)
3. **Pelletvorrat?** → [Pelletslager](../heizung/pelletslager.md) prüfen
4. **Pufferspeicher-Temperatur** prüfen

## Häufige Ursachen

| Symptom | Wahrscheinliche Ursache |
|---------|------------------------|
| Heizung startet nicht | Stromausfall, Pelletmangel, Verstopfung |
| Heizung läuft, Räume bleiben kalt | Umwälzpumpe defekt, Heizkurve falsch, Luft im System |
| Fehlercode am Display | siehe [Störungen](../heizung/stoerungen.md) |
| Rauchgeruch | **SOFORT abschalten + Lüften + Installateur!** |

## Notlösungen für Gäste

Bei längerem Ausfall (über 2-3h) im Winter:

- **Elektroheizlüfter** als Notlösung (Lager: *hier eintragen wo*)
- Gäste informieren und ggf. Entschädigung anbieten
- Bei längerem Ausfall: Gäste in andere Apartments umziehen oder Hotel anbieten

!!! warning "Frostgefahr"
    Bei Außentemperaturen unter 0°C und Heizungsausfall droht **Frostschaden an Wasserleitungen**. Bei Ausfall über 12h: alle Wasserleitungen entleeren in Erwägung ziehen.

## Fachfirma rufen

Siehe [Notfall-Kontakte](kontakte.md):

- Heizungsinstallateur (möglichst der Errichter der Anlage)
- Schornsteinfeger (bei Verdacht auf Abgasproblem)
