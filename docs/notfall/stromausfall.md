# Stromausfall

## Sofort prüfen

1. **Ist nur ein Apartment betroffen oder das ganze Haus?**
2. **Sind Nachbarhäuser auch betroffen?** → dann großflächiger Ausfall (Netzbetreiber)

## Nur ein Apartment betroffen

➡️ Wahrscheinlich Sicherung im **Unterverteiler des Apartments** ausgelöst.

1. Unterverteiler des Apartments öffnen (Standort siehe [Unterverteiler Apartments](../strom/unterverteiler-apt.md))
2. **FI-Schalter** prüfen (steht in Mittelstellung wenn ausgelöst → ganz nach unten, dann nach oben)
3. **Ausgelöste Sicherung** identifizieren (Hebel unten statt oben)
4. Ggf. den verursachenden Verbraucher (Wasserkocher, Föhn, etc.) abstecken
5. Sicherung wieder einschalten

!!! warning "Wenn FI sofort wieder auslöst"
    Es liegt ein Fehler vor (Defektes Gerät, Feuchtigkeit). Alle Verbraucher abstecken, Sicherung einzeln zuschalten, um Fehlerquelle zu finden. Im Zweifel → [Elektriker](kontakte.md).

## Ganzes Haus betroffen

➡️ Hauptverteiler prüfen.

1. **Hauptverteiler** öffnen (Standort siehe [Hauptverteiler](../strom/hauptverteiler.md))
2. Hauptschalter / FI-Schalter prüfen
3. Wenn Hauptschalter aus → vorsichtig wieder einschalten

## Großflächiger Stromausfall (mehrere Häuser)

1. **Netzbetreiber TINETZ** kontaktieren: ☎️ **0800 818 819** (24/7 Störungsnummer)
2. **Gäste informieren** – ggf. Taschenlampen / Kerzen anbieten
3. **Heizung beachten**: Pelletheizung läuft nicht ohne Strom → bei längerem Ausfall Frostgefahr (besonders im Winter!)

!!! info "Was funktioniert ohne Strom nicht"
    - Pelletheizung
    - Warmwasser-Boiler
    - WLAN / Internet
    - Schlüsselsafes mit elektronischem Code (falls vorhanden)
    - Kühl-/Gefrierschränke (max. 4-6h problemlos)

## Notfall-Ausstattung

- [ ] Taschenlampen im Eingangsbereich
- [ ] Reservebatterien
- [ ] Kerzen + Feuerzeug
- [ ] Powerbank für Mobiltelefone

## Nach dem Ausfall

- [ ] Heizung kontrollieren – läuft sie wieder an?
- [ ] Boiler prüfen – Warmwasser wieder verfügbar?
- [ ] Router / WLAN neu starten falls nötig
- [ ] Uhren in Apartments stellen
- [ ] Gefrierschrank-Inhalte kontrollieren
