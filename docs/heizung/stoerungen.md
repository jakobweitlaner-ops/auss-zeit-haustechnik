# Störungen & Fehlercodes

!!! info "Hinweis"
    Die exakten Fehlercodes hängen vom verwendeten Heizungsmodell ab. Diese Tabelle nach Bedarf mit den Codes aus dem Handbuch füllen.

## Häufige Fehlerbilder

### Heizung startet nicht

| Mögliche Ursache | Lösung |
|------------------|--------|
| Strom weg | Sicherung / FI prüfen |
| Pellet-Vorrat leer | Nachfüllen |
| Förderschnecke blockiert | Vorsichtig freilegen |
| Reset notwendig | Anlage 30s stromlos schalten |

### Anlage geht ständig auf Störung

→ Wartungstermin nötig, Fachbetrieb rufen.

### Schlechte Verbrennung (viel Asche, dunkler Rauch)

| Mögliche Ursache | Lösung |
|------------------|--------|
| Schlechte Pelletqualität | Lieferanten wechseln, ENplus A1 verwenden |
| Brennraum verschmutzt | Reinigung im Rahmen Wartung |
| Falsche Luftzufuhr | Fachbetrieb |

## Eskalation

1. Display-Reset versuchen
2. Anlage stromlos schalten (30s), dann wieder ein
3. Wenn Fehler bleibt → [Heizungsinstallateur](../notfall/kontakte.md)
