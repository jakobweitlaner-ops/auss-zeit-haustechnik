# Wartung Heizung

## Regelmäßige Aufgaben

### Wöchentlich (Heizsaison)

- [ ] Pelletvorrat kontrollieren
- [ ] Display auf Fehlermeldungen prüfen
- [ ] Kurzer Sichtkontrolle Heizraum (Geruch, ungewöhnliche Geräusche)

### Monatlich (Heizsaison)

- [ ] Aschelade leeren
- [ ] Brennraum visuell prüfen
- [ ] Pufferspeichertemperaturen prüfen (Schichtung ok?)

### Jährlich (Pflicht!)

- [ ] **Hauptwartung durch Fachbetrieb** *(Termin im Sommer/Herbst)*
- [ ] **Schornsteinfeger** Kehrtermin *(meist 2x/Jahr)*
- [ ] Sicherheitsventile prüfen
- [ ] Anlagendruck im Heizsystem prüfen (Soll: 1-2 bar)
