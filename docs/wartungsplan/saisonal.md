# Saisonale Aufgaben

## Vor Sommersaison (Mai)

- [ ] Apartments Frühjahrsputz
- [ ] Terrassen / Balkone reinigen
- [ ] Außenmöbel aufstellen
- [ ] Garten / Pizzaofen Apt. 5 prüfen
- [ ] Klimageräte (falls vorhanden) Filter reinigen

## Vor Wintersaison (Oktober)

- [ ] Pelletvorrat aufgefüllt
- [ ] Heizung Funktionsprüfung
- [ ] Außenwasser entleert (siehe [Frostschutz](../wasser/frostschutz.md))
- [ ] Schneeräumgerät / Streumittel bereit
- [ ] Winterreifen Fahrzeuge
- [ ] Skiraum / Trockenraum hergerichtet
